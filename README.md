# Homebrew Tap for git-ai

AI-powered Git commit grouping and message generation.

## Installation

```bash
brew tap ertiz82/gitai
brew install git-ai
```

## Configuration

Set your API key:

```bash
export CLOUD_AI_API_KEY="your-api-key"
```

Or create `jira.local.json` in your project root.

## Usage

```bash
cd your-git-repo
git-ai commit
```
